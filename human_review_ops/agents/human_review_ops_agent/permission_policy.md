# 权限策略

## 默认权限

- 默认只读。
- 默认不发送真实通知。
- 默认不写入线上状态。
- 默认不修改业务配置。

## 允许

- 读取本地场景包。
- 读取 Skill 内调试快照。
- 生成 QueryPlan。
- 生成 source_footer。
- 调用 mock / 只读预检 Tool 并生成 `tool_call_record`。
- 在 QueryPlan 通过断言、数据源属于允许来源、工具权限为只读时，执行治理数据源的只读查询。
- 生成通知草稿。
- 记录调试检查清单。

## 需要人工确认

- 发送飞书消息。
- 写入 Lark Base 或其他状态存储。
- 创建或更新工单。
- 覆盖标准样本池或绕过默认过滤条件。
- 查询未治理来源、禁用来源或包含敏感明细的数据。
- 执行非白名单 CLI 或可能产生副作用的查询/导出。

## 禁止

- 自动审批。
- 自动变更策略配置。
- 自动删除或覆盖场景包。
- 输出未脱敏的真实线上数据明细。
